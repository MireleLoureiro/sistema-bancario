# Sistema Bancário

Sistema bancário desenvolvido em Java puro para praticar conceitos de Orientação a Objetos, Coleções e Streams.

## Funcionalidades
- Cadastro de contas correntes e poupança
- Depósito, saque e transferência entre contas
- Relatórios com streams (saldo total, maior saldo, titulares ordenados)

## Tecnologias
- Java 17+

## Como executar
1. Clone o repositório
2. Abra na sua IDE (IntelliJ, Eclipse)
3. Execute a classe `Main.java`
